Política de Privacidade

Última atualização: [DATA ATUAL - ex: 15 de Junho de 2026]

Coleta de Dados

Esta extensão NÃO coleta, armazena, processa ou transmite nenhum dado do usuário.

O que NÃO coletamos

  • Não coletamos informações pessoais (nome, e-mail, idade, etc.)
  • Não coletamos histórico de navegação
  • Não coletamos senhas ou credenciais
  • Não coletamos informações de localização
  • Não coletamos dados de saúde
  • Não coletamos informações financeiras ou de pagamento
  • Não coletamos comunicações pessoais (e-mails, mensagens)
  • Não coletamos atividade do usuário (cliques, rolagem, etc.)
  • Não coletamos conteúdo de sites visitados

Como a extensão funciona

A extensão opera inteiramente de forma local no navegador do usuário. Ela só é ativada quando o usuário clica no ícone da extensão, e apenas remove elementos visuais (pop-ups, banners, modais) da página atual. Nenhum dado é enviado para servidores externos.

Permissões utilizadas

As permissões activeTab e scripting são usadas exclusivamente para permitir que o usuário remova elementos da página ativa mediante clique manual. Estas permissões NÃO são usadas para coletar dados.

Conformidade com as políticas do Chrome Web Store

O uso das informações segue as diretrizes da Chrome Web Store User Data Policy, incluindo os requisitos de Limited Use. Nenhum dado é transferido a terceiros, usado para anúncios personalizados ou para determinar crédito.
