🔒 Política de Privacidade

Clean Page - Removedor de Pop-ups

🚫 ZERO COLEÇÃO DE DADOS
📅 Última atualização: 15 de Junho de 2026

Nenhum dado é coletado

Esta extensão NÃO coleta, armazena, processa ou transmite nenhum dado do usuário. Sua privacidade é 100% respeitada.

Sem cadastro Sem login Sem servidores Sem rastreamento

O que NÃO coletamos

  • Informações pessoais (nome, e-mail, idade, etc.)
  • Histórico de navegação
  • Senhas ou credenciais
  • Informações de localização
  • Dados de saúde
  • Informações financeiras ou de pagamento
  • Comunicações pessoais (e-mails, mensagens)
  • Atividade do usuário (cliques, rolagem, etc.)
  • Conteúdo de sites visitados
  • Endereço de IP
  • Cookies ou identificadores

Como a extensão funciona

A extensão opera inteiramente de forma local no navegador do usuário. Ela só é ativada quando você clica no ícone da extensão, e apenas remove elementos visuais (pop-ups, banners, modais) da página atual.

📌 Importante: Nenhum dado é enviado para servidores externos. Tudo acontece exclusivamente no seu computador.

Permissões utilizadas

As permissões activeTab e scripting são usadas exclusivamente para:

  • ✓ Acessar APENAS a aba onde você clicou
  • ✓ Injetar código para remover elementos visuais
  • ✓ Funcionar apenas mediante clique manual

Estas permissões NÃO são usadas para coletar dados. Elas são necessárias apenas para que a extensão execute sua função principal.

Conformidade com políticas

O uso das informações segue as diretrizes da Chrome Web Store User Data Policy, incluindo os requisitos de Limited Use. Nenhum dado é:

  • ✓ Transferido a terceiros
  • ✓ Usado para anúncios personalizados
  • ✓ Usado para determinar crédito
  • ✓ Compartilhado com qualquer serviço externo
🔒 Compromisso com a privacidade: "The use of information received from Google APIs will adhere to the Chrome Web Store User Data Policy, including the Limited Use requirements."
